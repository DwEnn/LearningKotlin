## Basic Syntax

1. 패키지 정의
 - 패키지 정의는 파일 최상단에 위치
 - 디렉터리와 패키지를 일치시키지 않아도 됨

1. 함수 정의
 - 함수는 *fun* 키워드로 정의
 - 함수 몸체가 식(expression)인 경우 return 생략 가능
 - **fun** sum(a: Int, b: Int) = a + b
